# 2026-02-05 — Landing Page look.me

## Sessão

Início do desenvolvimento da landing page B2B para o look.me.

## Pesquisa

### Referências analisadas
1. **Seedream (ByteDance)** — dark theme, animações suaves, antes/depois
2. **Relats Top Tier** — estrutura corporativa
3. **Doris.AI** — concorrente B2C direto, valida mercado

### Descobertas
- Doris.AI já faz virtual try-on B2C (vende direto pra consumidor)
- Nosso diferencial: B2B (vender pra lojas de e-commerce)
- Dark theme reforça posicionamento premium

## Decisões Tomadas

### Identidade
- **Marca:** look.me
- **Slogan:** "O provador oficial do seu e-commerce."
- **Paleta:** Dark theme (#0A0A0B background, #141416 cards, #6366F1 accent)

### Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Deploy: Vercel (static export)

### Estrutura da Landing
1. Hero — proposta de valor + CTA
2. Problem — dor do lojista (devolução, conversão baixa)
3. Solution/Demo — antes/depois, vídeo do produto
4. Benefits — 3 cards (conversão, devolução, engajamento)
5. How it Works — 3 steps de integração
6. Integrations — logos (Shopify, WooCommerce, VTEX, Nuvemshop)
7. CTA — chamada final
8. Footer

### Mensagens-chave B2B
- "Aumente conversão. Reduza devolução."
- "Integração em menos de 10 minutos"
- Foco em ROI pro lojista

## Arquivos Criados

### Config
- `package.json` — deps (next, react, framer-motion, lucide-react)
- `next.config.js` — static export habilitado

- `postcss.config.js`

### App
- `src/app/globals.css` — animações, scrollbar, gradientes
- `src/app/layout.tsx` — metadata, fonts (Inter + Instrument Sans)
